🧪 1. Обновление pip
python -m pip install --upgrade pip

🌀 2. Создание и активация виртуального окружения
Windows:
python -m venv venv
venv\Scripts\activate

Linux/macOS:
python3 -m venv venv
source venv/bin/activate

📦 3. Установка зависимостей из requirements.txt
bash
pip install -r requirements.txt
